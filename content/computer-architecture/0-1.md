## Course Description
본 스터디에서는 컴퓨터 시스템이 어떻게 구성되어있고 디자인되어있는지 공부합니다. </br> 특히 CPU, memory를 포함한 하드웨어 시스템에 적용되고 있는 디자인 특성들을 알아봅니다.


## Prerequisites and Co-requisites
C 프로그래밍, 논리회로, 어셈블리어를 알면 더욱 좋습니다.
